## Comment rejoindre le programme d'affiliation

Suivez ces étapes pour rejoindre le programme d'affiliation Ollabot :

### Étape 1 : Visitez la page du programme d'affiliation

Visitez ce lien : [ollabot.promotekit.com](https://ollabot.promotekit.com)

### Étape 2 : Inscrivez-vous
Inscrivez-vous en utilisant votre email. 

![Inscription d'affiliation](/images/affiliate/affiliate_signup.png)

Vous recevrez un code de vérification que vous devrez entrer pour compléter l'inscription.

![Code d'affiliation](/images/affiliate/affiliate_code.png)

### Étape 3 : Remplissez le formulaire

Après votre inscription, remplissez le formulaire avec les informations requises, y compris votre email PayPal pour recevoir les paiements.

![Affiliate Form](/images/affiliate/affiliate_form.png)

### Étape 4 : Accédez au tableau de bord

![Tableau de bord d'affiliation](/images/affiliate/affiliate_dashboard.png)

Une fois que vous avez rempli le formulaire, vous serez redirigé vers le tableau de bord où vous pourrez voir votre lien d'affiliation.

## Conclusion

En suivant ces étapes, vous pouvez facilement rejoindre le programme d'affiliation Ollabot et commencer à gagner des commissions en référant de nouveaux clients.
