---
title: Using Ollabot API
description: This page will help you get started with using the API of Ollabot.
---
Ollabot offers an API for you to interact with your chatbots if you are subscribed to a paid plan. To get an API key, follow the steps below:

1. Go to [https://app.ollabot.com/dashboard/settings](https://app.ollabot.com/dashboard/settings) and scroll to the bottom.

<Image
  src="/images/docs/api_key.png"
  width="718"
  height="404"
  alt="Image"
/>

2. Pressing the eye icon will reveal the secret API key. Save it somewhere safe. This is the API key that you will use to authenticate your requests to the Ollabot API.

<Image
  src="/images/docs/api_eye_icon.png"
  width="718"
  height="404"
  alt="Image"
/>

3. To get the ID of a created Chatbot, navigate to [https://app.ollabot.com/dashboard](https://app.ollabot.com/dashboard) and click on the Chatbot you would like to get the ID for.

<Image
  src="/images/docs/bot_id.png"
  width="718"
  height="404"
  alt="Image"
/>

You can also quickly get the ID by using the URL of the webpage. Once you have clicked on the Chatbot, the URL will contain your chat ID in the form `https://app.ollabot.com/bot/<Your Chatbot ID>`.
