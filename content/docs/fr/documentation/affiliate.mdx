## Comment rejoindre le programme d'affiliation

Suivez ces étapes pour rejoindre le programme d'affiliation Ollabot :

### Étape 1: Visitez la page du programme d'affiliation

Visitez ce lien : [ollabot.promotekit.com](https://ollabot.promotekit.com)

### Étape 2: Inscrivez-vous

Inscrivez-vous en utilisant votre adresse e-mail.

![Inscription à l'affiliation](/images/affiliate/affiliate_signup.png)

Vous recevrez un code de vérification, que vous devrez saisir pour finaliser le processus d'inscription.

![Code d'affiliation](/images/affiliate/affiliate_code.png)

### Étape 3: Remplissez le formulaire

Après vous être inscrit, remplissez le formulaire avec les détails requis, y compris votre adresse e-mail PayPal pour recevoir les paiements.

![Formulaire d'affiliation](/images/affiliate/affiliate_form.png)

### Étape 4: Accédez au tableau de bord

![Tableau de bord de l'affiliation](/images/affiliate/affiliate_dashboard.png)

Une fois le formulaire rempli, vous serez redirigé vers le tableau de bord où vous pourrez voir votre lien d'affiliation.

## Conclusion

En suivant ces étapes, vous pouvez facilement rejoindre le programme d'affiliation Ollabot et commencer à gagner des commissions en recommandant de nouveaux clients.
