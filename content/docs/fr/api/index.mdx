---
title: Utilisation de l'API Ollabot
description: Cette page vous aidera à démarrer avec l'utilisation de l'API d'Ollabot.
---

Ollabot offre une API pour interagir avec vos chatbots si vous êtes abonné à un plan payant. Pour obtenir une clé API, suivez les étapes ci-dessous :

1. Allez sur [https://app.ollabot.com/dashboard/settings](https://app.ollabot.com/dashboard/settings) et faites défiler jusqu'en bas.

<Image
  src="/images/docs/api_key.png"
  width="718"
  height="404"
  alt="Image"
/>

2. Appuyez sur l'icône de l'œil pour révéler la clé API secrète. Enregistrez-la dans un endroit sûr. C'est la clé API que vous utiliserez pour authentifier vos requêtes à l'API Ollabot.

<Image
  src="/images/docs/api_eye_icon.png"
  width="718"
  height="404"
  alt="Image"
/>

3. Pour obtenir l'ID d'un chatbot créé, accédez à [https://app.ollabot.com/dashboard](https://app.ollabot.com/dashboard) et cliquez sur le chatbot pour lequel vous souhaitez obtenir l'ID.

<Image
  src="/images/docs/bot_id.png"
  width="718"
  height="404"
  alt="Image"
/>

Vous pouvez également obtenir rapidement l'ID en utilisant l'URL de la page Web. Une fois que vous avez cliqué sur le chatbot, l'URL contiendra l'ID de votre chatbot sous la forme `https://app.ollabot.com/bot/<Your Chatbot ID>`.
