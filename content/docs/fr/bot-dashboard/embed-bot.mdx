Ce guide fournit des instructions pour intégrer le chatbot sur votre site web en utilisant le code de script fourni.

## Intégration avec Script

Pour ajouter une bulle de chat dans le coin inférieur droit de votre site web, suivez ces étapes :

### Étape 1 : Intégrer le Script

Copiez et collez le script suivant dans la section `<head>` ou `<body>` de votre fichier HTML. Cela ajoutera la bulle de chat à votre site web.

```html
<script>
window.embeddedChatbotConfig = {
domain: "https://app.ollabot.com",
botId: "VOTRE_ID_DE_BOT",
};
</script>
<script 
src="https://app.ollabot.com/embed.min.js">
</script>
```

Remplacez `"VOTRE_ID_DE_BOT"` par votre véritable ID de bot.

### Étape 2 : Confirmation de l'Intégration

Une fois que vous avez ajouté le code de script à votre HTML, votre chatbot sera intégré et devrait apparaître sur votre site web.

## Conclusion

L'intégration du chatbot sur votre site web vous permet d'interagir avec vos visiteurs et de fournir un support en temps réel. Suivez les étapes ci-dessus pour intégrer le chatbot de manière transparente.
