Ce guide vous guidera à travers les étapes pour gérer la base de connaissances de votre chatbot en ajoutant des fichiers et des liens. Ces sources seront scrutées et ajoutées à la base de données de votre bot pour améliorer ses réponses et ses capacités.

## Ajout de Sources de Connaissances

Vous pouvez enrichir la base de connaissances de votre chatbot en ajoutant divers types de fichiers et en incluant des liens. Les formats de fichier pris en charge comprennent DOC, MD, CSV et TXT.

<Image src="/images/docs/dashboard-upsertfiles.png" width="718" height="404" alt="Gérer la Base de Connaissances" />

### Ajout de Fichiers

Suivez ces étapes pour télécharger des fichiers dans la base de connaissances de votre bot :

1. **Connectez-vous à Ollabot:** Commencez par vous connecter à votre compte Ollabot. Accédez au tableau de bord du bot.
2. **Sélectionnez Votre Bot:** Choisissez le bot que vous souhaitez mettre à jour.
3. **Accédez à la Section Base de Connaissances:** Dans le tableau de bord du bot, trouvez la section "Base de Connaissances".
4. **Téléchargez des Fichiers:** Cliquez sur le bouton "Télécharger des Fichiers". Vous pouvez télécharger des fichiers aux formats DOC, MD, CSV ou TXT.

### Exemples de Téléchargement de Fichiers

- **Fichiers DOC:** Téléchargez des documents Word contenant des informations précieuses.
- **Fichiers MD:** Utilisez des fichiers Markdown pour des données structurées.
- **Fichiers CSV:** Ajoutez des fichiers CSV avec des données tabulaires.
- **Fichiers TXT:** Incluez des fichiers texte simples pour des données textuelles simples.

### Ajout de Liens

Vous pouvez également ajouter des URL de pages web contenant des informations utiles pour votre chatbot. Le contenu de ces liens sera scruté et ajouté à la base de connaissances du bot. La limite totale pour cela est de 50 liens.

1. **Accédez à Ajouter des Liens:** Dans la section Base de Connaissances, trouvez le bouton "Ajouter des Liens".
2. **Explorer les URLs:** Vous pouvez ajouter plusieurs liens en explorant une URL spécifique. La limite pour cela est de 50 liens.

3. **Entrez d'autres URLs:** Saisissez les URLs d'autres pages web que vous souhaitez ajouter.
4. **Scruter le Contenu:** Le système scrutera le contenu de l'URL et l'intégrera à la base de connaissances de votre bot.

### Formats de Fichier Supportés

| Format de Fichier | Description                       |
|-------------------|-----------------------------------|
| **DOC**           | Documents Microsoft Word          |
| **MD**            | Fichiers Markdown                 |
| **CSV**           | Fichiers de valeurs séparées par virgules |
| **TXT**           | Fichiers texte simples            |

## Gestion et Mise à Jour des Connaissances

Mettre régulièrement à jour la base de connaissances de votre bot garantit qu'elle reste précise et pertinente. Vous pouvez :

- **Mettre à Jour les Fichiers Existant:** Re-téléchargez les versions mises à jour des fichiers existants.
- **Ajouter de Nouvelles Sources:** Ajoutez continuellement de nouveaux fichiers et liens à mesure que vos informations évoluent.
- **Supprimer les Informations Obsolètes:** Supprimez les fichiers et liens qui ne sont plus pertinents.

## Tester Votre Bot

Après avoir mis à jour la base de connaissances, il est crucial de tester votre bot pour vous assurer qu'il utilise correctement les nouvelles informations. Vous pouvez tester votre bot en interagissant directement avec l'interface de chat à `/customization`.

<Image src="/images/docs/customization_chat.png" width="718" height="404" alt="Tester le Bot" />

1. **Interagir avec le Bot:** Engagez-vous dans diverses conversations pour voir comment le bot répond en utilisant les nouvelles connaissances.
2. **Surveiller les Réponses:** Vérifiez l'exactitude et la pertinence des réponses du bot.
3. **Ajuster au Besoin:** Si les réponses du bot ne sont pas satisfaisantes, apportez les ajustements nécessaires à la base de connaissances.

## Conclusion

La gestion de la base de connaissances de votre chatbot est essentielle pour maintenir son efficacité et sa fiabilité. En ajoutant des fichiers et des liens, vous pouvez vous assurer que votre bot a accès aux informations les plus précises et les plus à jour.

Utilisez ces fonctionnalités pour garder votre bot bien informé et capable de gérer une large gamme de requêtes d'utilisateurs, améliorant ainsi l'expérience utilisateur globale.
