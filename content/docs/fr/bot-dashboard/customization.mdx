Ce guide vous guidera à travers les différentes options disponibles pour personnaliser votre chatbot sur Ollabot.

## Modifier la boîte de chat

Lorsque vous accédez pour la première fois à la page de personnalisation, vous verrez les options de personnalisation de la boîte de chat. Cliquez d'abord sur le bouton **Modifier** pour commencer à personnaliser votre boîte de chat.

<Image src="/images/bot-dashboard/customization-edit.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Changer le **Nom du Bot**, le **Nom de l'entreprise**, le **Logo de l'entreprise** et le **Logo de l'avatar** mettra à jour la fenêtre de chat en temps réel.

<Image src="/images/docs/customize_chatbox.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Vous pouvez personnaliser la boîte de chat en modifiant les éléments suivants :

### Logo de l'entreprise

Vous pouvez télécharger un **Logo de l'entreprise** à afficher dans la fenêtre de chat. Le logo aide à la création de la marque et donne aux utilisateurs un identifiant visuel de votre entreprise.

### Nom de l'entreprise

Le **Nom de l'entreprise** est affiché dans la fenêtre de chat et aide les utilisateurs à identifier l'organisation derrière le chatbot. Assurez-vous qu'il reflète fidèlement votre entreprise.

### Nom du Bot

Le **Nom du Bot** est le nom affiché aux utilisateurs lorsqu'ils interagissent avec votre chatbot. Choisissez un nom qui représente votre marque ou votre service.

### Logo de l'avatar

L'**Image de l'avatar** est l'image représentant le chatbot. Vous pouvez télécharger un avatar qui correspond à la personnalité du bot.

### Couleur d'accent

La **Couleur d'accent** définit la couleur d'accent de la fenêtre de chat. Utilisez un code couleur HEX pour spécifier la couleur.

## Modifier les paramètres du chatbot

<Image src="/images/docs/customize_chat.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

### Description

La **Description** fournit aux utilisateurs un aperçu des capacités de votre chatbot. Cela peut inclure le but du bot, ses capacités et toute autre information pertinente.

### Message de bienvenue

Le **Message de bienvenue** est le message initial que les utilisateurs verront lorsqu'ils commenceront à discuter avec votre bot. Rendez-le accueillant et informatif.

### Créativité

Le réglage de la **Créativité** contrôle la créativité des réponses du chatbot. Il varie de 0 à 1, les valeurs plus élevées rendant les réponses du bot plus variées et créatives.

### Ton

Le **Ton** du chatbot définit sa personnalité et son style de communication. Les valeurs possibles sont :
- Professionnel/Académique
- Décontracté/Amical
- Instructif/Autoritaire
- Conversationnel/Engageant
- Inspirant/Motivant

### Inclure le filigrane

L'option **Inclure le filigrane** vous permet d'ajouter un filigrane à la fenêtre de chat. Cette fonctionnalité est disponible uniquement pour les utilisateurs Illimités et Entreprise.

## Tester le chatbot

Vous pouvez tester votre chatbot en cliquant sur la fenêtre contextuelle. Cela ouvrira la fenêtre de chat, où vous pourrez interagir avec votre bot et voir les changements en temps réel.

<Image src="/images/docs/chat_test.png" width="718" height="404" alt="Chat Test" />

## Sauvegarder vos modifications

Vous pouvez sauvegarder vos modifications en appuyant sur **Sauvegarder les modifications**. Si vous actualisez la page, vous retrouverez vos modifications enregistrées.

<Image src="/images/bot-dashboard/saved-settings.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Cette section explique comment vous pouvez voir l'effet de vos personnalisations en temps réel :

- **Icône de la bulle de chat** : Cliquez sur l'icône de chat pour commencer une discussion et voir l'icône de la bulle de chat sélectionnée en action.
- **Couleur d'arrière-plan** : La couleur d'arrière-plan que vous définissez sera appliquée à la fenêtre de chat.
- **Message de bienvenue** : Le chatbot affichera le message de bienvenue que vous avez configuré.
- **Logo de l'entreprise** : Le logo sera affiché dans la fenêtre de chat.
- **Image de l'avatar** : L'avatar représentera le chatbot dans la fenêtre de chat.
- **Inclure le filigrane** : Si activé, le filigrane sera visible dans la fenêtre de chat.

## Conclusion

Personnaliser votre chatbot aide à créer une expérience utilisateur unique qui correspond à votre marque. Utilisez les options disponibles pour personnaliser votre bot et le rendre plus engageant pour vos utilisateurs.
