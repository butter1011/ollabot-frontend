Ce guide vous expliquera les étapes pour supprimer votre chatbot sur Ollabot. Supprimer un chatbot est un processus simple, mais il est important de s'assurer que vous souhaitez vraiment le supprimer, car cette action est irréversible.

## Supprimer un chatbot

Vous pouvez supprimer votre chatbot en suivant ces étapes. Assurez-vous de confirmer la suppression en tapant la phrase de confirmation requise.

### Étapes pour supprimer votre chatbot

1. **Connectez-vous à Ollabot** : Commencez par vous connecter à votre compte Ollabot. Naviguez jusqu'au tableau de bord du bot.
2. **Sélectionnez votre bot** : Choisissez le bot que vous souhaitez supprimer.
3. **Accédez à la section Supprimer le bot** : Dans le tableau de bord du bot, trouvez la section "Supprimer le bot".

### Confirmer la suppression

Pour s'assurer que le processus de suppression est intentionnel et pour éviter les suppressions accidentelles, vous devrez confirmer votre action en tapant la phrase de confirmation.

1. **Tapez la phrase de confirmation** : Pour confirmer la suppression, tapez la phrase "Supprimer le chatbot" dans le champ de texte prévu à cet effet.

<Image src="/images/bot-dashboard/delete-chatbot.png" width="718" height="404" alt="Confirm Delete Chatbot" />

### Finaliser la suppression

Une fois que vous avez entré les informations requises, procédez à la suppression :

1. **Cliquez sur le bouton Supprimer** : Cliquez sur le bouton "Supprimer le chatbot" pour continuer.
2. **Attendez la confirmation** : Une notification apparaîtra confirmant que le chatbot a été supprimé avec succès.

## Considérations importantes

Avant de supprimer votre chatbot, prenez en compte les éléments suivants :

- **Action irréversible** : Supprimer un chatbot est permanent et ne peut pas être annulé. Assurez-vous d'avoir sauvegardé toutes les données importantes.
- **Impact sur les utilisateurs** : Si votre chatbot est activement utilisé par des clients ou des utilisateurs, assurez-vous que sa suppression ne perturbera pas leur expérience.

## Conclusion

Supprimer un chatbot sur Ollabot est une action simple mais significative. Assurez-vous d'avoir pris en compte toutes les implications avant de continuer. En suivant ce guide, vous pouvez vous assurer que le processus de suppression est géré de manière fluide et efficace.

Si vous rencontrez des problèmes ou avez besoin d'aide supplémentaire, n'hésitez pas à contacter le support Ollabot pour obtenir de l'aide.

Utilisez ces étapes pour gérer votre chatbot efficacement, assurant une expérience fluide et sans accroc pour vos utilisateurs.
