## Étape 1: Configurer Votre Chatbot Ollabot

Pour intégrer votre chatbot AI Ollabot dans votre application Bubble, commencez par vous connecter à votre compte Ollabot. Si vous n'avez pas encore de compte, vous pouvez vous inscrire gratuitement. Après vous être connecté, configurez votre bot sur la plateforme Ollabot en téléchargeant des sources de données pertinentes telles que des fichiers, des extraits de texte, des sites web ou des paires de questions-réponses pour construire la base de connaissances du chatbot.

Suivez cette feuille de route étape par étape pour déployer avec succès votre chatbot Ollabot.

## Étape 2: Générer et Copier le Code d'Intégration de Votre Chatbot Ollabot

1. Chaque chatbot créé sur Ollabot dispose d'un code d'intégration unique. Une fois votre chatbot configuré sur Ollabot, accédez à votre page de tableau de bord et sélectionnez le bot spécifique que vous souhaitez intégrer à votre application Bubble.

<Image src="/images/integrations/dashboard.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>
2. En cliquant sur le bot, vous serez redirigé vers sa page de prévisualisation. À partir de là, allez à l'onglet "Intégrer sur le Site" et cliquez sur le bouton "Rendre Public".

3. Une fenêtre contextuelle apparaîtra avec le code HTML fourni. Cliquez sur "Copier le Script" pour copier le snippet de code.

<Image src="/images/integrations/embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

## Étape 3: Intégrer le Chatbot Ollabot dans Votre Application Bubble

1. Après avoir copié votre code d'intégration Ollabot, connectez-vous à votre compte Bubble et accédez à votre tableau de bord.

2. Sur votre tableau de bord, sélectionnez l'application ou le site Bubble sur lequel vous souhaitez intégrer le chatbot, puis cliquez sur le bouton "Lancer l'Éditeur" à côté.

<Image src="/images/integrations/bubble-app.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

3. Lorsque l'Éditeur Bubble s'ouvre, faites défiler jusqu'à la section de la page où vous souhaitez ajouter le code d'intégration.

4. Dans la barre latérale gauche de l'éditeur, trouvez le composant HTML et faites-le glisser vers la section désirée de la page.

<Image src="/images/integrations/bubble-html.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

5. Double-cliquez sur le composant HTML pour ouvrir l'éditeur de code.

6. Collez le code d'intégration dans l'éditeur, et vous devriez voir une icône de chatbot flottante apparaître dans le coin inférieur gauche du canevas d'édition.

<Image src="/images/integrations/bubble-embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

7. Prévisualisez votre application Bubble pour tester votre chatbot.

<Image src="/images/integrations/bubble-chat.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Félicitations, votre chatbot Ollabot est maintenant opérationnel sur votre application Bubble !

**Remarque** : Vous pouvez personnaliser l'apparence et les couleurs de votre bot sur votre tableau de bord Ollabot. Accédez à votre tableau de bord, choisissez un bot, cliquez sur l'onglet "Paramètres" en haut de la page, puis cliquez sur "Interface de Chat" dans la barre latérale gauche pour accéder aux options de personnalisation du chatbot.
