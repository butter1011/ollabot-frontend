## Étape 1 : Connectez-vous à votre compte Ollabot et configurez votre bot

Pour intégrer votre bot Ollabot sur votre site Weebly, commencez par vous connecter à votre compte Ollabot. Si vous n'avez pas encore de compte, vous pouvez en créer un gratuitement. Une fois connecté, vous pouvez configurer votre bot dans Ollabot en téléchargeant des sources de données comme des fichiers, des extraits de texte, des sites web ou des paires de questions-réponses à partir desquelles le bot peut apprendre. Voici un guide étape par étape pour configurer votre chatbot Ollabot.

## Étape 2 : Générez et copiez le code d'intégration de votre chatbot Ollabot

1. Chaque chatbot créé sur Ollabot dispose d'un code d'intégration unique. Une fois votre chatbot configuré sur Ollabot, accédez à votre page de tableau de bord et sélectionnez le bot spécifique que vous souhaitez intégrer à votre application Bubble.

<Image src="/images/integrations/dashboard.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>
2. En cliquant sur le bot, vous accéderez à sa page de prévisualisation. De là, allez à l'onglet "Intégrer sur le site" et cliquez sur le bouton "Rendre public".

3. Une fenêtre contextuelle apparaîtra avec le code HTML fourni. Cliquez sur "Copier le script" pour copier le snippet de code.

<Image src="/images/integrations/embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

## Étape 3 : Ajoutez le code d'intégration à votre site Weebly

Pour ajouter un chatbot Ollabot à votre site Weebly :

1. Connectez-vous à votre compte Weebly et accédez à la page "Modifier le site".

2. Sélectionnez la page à modifier.

3. Dans la barre d'outils de base sur le côté gauche de la page de modification, localisez le widget qui indique "Code d'intégration".

<Image src="/images/integrations/weebly-html.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

4. Faites glisser l'élément "Code d'intégration" à l'endroit où vous souhaitez que votre chatbot apparaisse sur la page.

5. Vous devriez voir un élément de texte indiquant "Cliquez pour définir HTML personnalisé" à l'endroit où le widget de code d'intégration a été placé. Cliquez sur le texte, puis sur "Modifier HTML personnalisé".

6. Collez votre code d'intégration Ollabot dans la boîte HTML personnalisée.

<Image src="/images/integrations/weebly-embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

7. Cliquez à

 l'extérieur de l'élément et votre chatbot devrait apparaître sur la page.

<Image src="/images/integrations/weebly-chat.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Félicitations, votre chatbot est désormais opérationnel sur votre site Weebly !

**Note** : Vous pouvez personnaliser l'apparence et les couleurs de votre bot sur votre tableau de bord Ollabot. Pour cela, accédez à votre tableau de bord, choisissez un bot, cliquez sur l'onglet "Paramètres" en haut de la page, puis sur "Interface de chat" dans la barre latérale gauche pour découvrir les options de personnalisation du chatbot.
