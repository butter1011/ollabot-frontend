## Étape 1: Connectez-vous à Ollabot et Configurez Votre Chatbot

Pour ajouter un chatbot Ollabot à votre site Shopify, connectez-vous à votre compte Ollabot pour créer et configurer un chatbot. Si vous n'avez pas de compte Ollabot, vous pouvez commencer par en créer un gratuitement. Si vous n'êtes pas sûr de savoir comment créer un chatbot, voici un guide détaillé sur la création d'un chatbot sur Ollabot.

## Étape 2: Générer et Copier le Code d'Intégration de Votre Chatbot O

llabot

1. Une fois votre chatbot configuré sur Ollabot, accédez à votre tableau de bord et sélectionnez le chatbot que vous souhaitez intégrer à votre site Shopify.

## Étape 3: Intégrer le Chatbot Ollabot dans Votre Site Shopify

1. Connectez-vous à votre tableau de bord Shopify et accédez à votre administrateur Shopify.

2. Dans le panneau d'administration Shopify, allez à la section "Online Store" (Boutique en ligne) et sélectionnez "Themes" (Thèmes).

3. Dans la page des thèmes, cliquez sur "Customize" (Personnaliser) pour le thème que vous utilisez actuellement.

4. Dans le menu de personnalisation du thème, trouvez ou créez un endroit approprié pour votre chatbot. Cela peut être dans le pied de page, le menu de navigation, ou une fenêtre contextuelle sur une page spécifique.

5. Pour ajouter le chatbot, choisissez l'option "Ajouter une section" ou "Ajouter un contenu personnalisé" selon le placement souhaité.

6. Sélectionnez "HTML" parmi les options de contenu disponible.

7. Collez le code d'intégration que vous avez copié à partir de votre tableau de bord Ollabot.

<Image src="/images/integrations/shopify-html.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

8. Enregistrez vos modifications et prévisualisez votre site Shopify pour vérifier que le chatbot Ollabot fonctionne comme prévu.

<Image src="/images/integrations/shopify-preview.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Félicitations ! Votre chatbot Ollabot est maintenant intégré et opérationnel sur votre site Shopify.

**Remarque** : Vous pouvez personnaliser l'apparence et le comportement de votre chatbot directement à partir de votre tableau de bord Ollabot. Accédez à votre tableau de bord, choisissez le chatbot que vous souhaitez personnaliser, cliquez sur l'onglet "Paramètres" en haut de la page, puis accédez à "Interface de Chat" dans la barre latérale gauche pour ajuster les paramètres selon vos préférences.
