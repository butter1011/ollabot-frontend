## Étape 1: Configuration de Votre Chatbot Ollabot

Pour intégrer votre chatbot Ollabot sur votre site Wix, commencez par vous connecter à votre compte Ollabot. Si vous n'avez pas encore de compte, vous pouvez vous inscrire gratuitement. Une fois connecté, créez et configurez votre chatbot en important des données pertinentes comme des fichiers, extraits de texte, ou des paires de questions-réponses pour construire la base de connaissances du chatbot.

Suivez cette feuille de route étape par étape pour déployer avec succès votre chatbot Ollabot.

## Étape 2: Générer et Copier le Code d'Intégration de Votre Chatbot Ollabot

1. Chaque chatbot créé sur Ollabot a un code d'intégration unique. Après avoir configuré votre chatbot sur Ollabot, accédez à votre tableau de bord et sélectionnez le chatbot que vous souhaitez intégrer à votre site Wix.

<Image src="/images/integrations/dashboard.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

2. Sur la page de prévisualisation du chatbot, cliquez sur l'onglet "Intégrer sur le Site". Vous verrez une fenêtre contextuelle avec le code HTML d'intégration. Cliquez sur "Copier le Script" pour copier le snippet de code.

<Image src="/images/integrations/embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

## Étape 3: Intégrer le Chatbot Ollabot dans Votre Site Wix

1. Connectez-vous à votre compte Wix et accédez à votre tableau de bord.

2. Sélectionnez le site Wix sur lequel vous souhaitez intégrer votre chatbot Ollabot.

3. Dans l'éditeur Wix, accédez à la page où vous souhaitez ajouter votre chatbot. Cliquez sur "Ajouter" > "Autres" > "HTML iframe".

<Image src="/images/integrations/wix-add.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

4. Une fenêtre contextuelle apparaîtra. Cliquez sur "Insérer un code" et collez le code d'intégration que vous avez copié depuis votre tableau de bord Ollabot.

<Image src="/images/integrations/wix-embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

5. Cliquez sur "Mettre à jour" pour appliquer les modifications. Vous verrez votre chatbot Ollabot apparaître sur votre site Wix.

<Image src="/images/integrations/wix-preview.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Félicitations ! Votre chatbot Ollabot est maintenant intégré et opérationnel sur votre site Wix.

**Remarque** : Pour personnaliser l'apparence et le comportement de votre chatbot, accédez à votre tableau de bord Ollabot, sélectionnez votre chatbot, cliquez sur l'onglet "Paramètres" en haut de la page, puis accédez à "Interface de Chat" dans la barre latérale gauche pour ajuster les paramètres selon vos préférences.
