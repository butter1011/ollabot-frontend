## Étape 1 : Configurez votre chatbot Ollabot

Pour intégrer votre chatbot Ollabot AI dans votre bot Telegram, commencez par générer votre jeton de bot Telegram.
Suivez cette feuille de route étape par étape pour déployer votre chatbot Ollabot de manière superaccessible.
1. Recherchez botFather dans Telegram. Envoyez "/start" pour commencer.

<Image src="/images/integrations/telegram-app1.png"
width="718"
height="404"
alt="Image"
/>
2. Saisissez "/newbot" et définissez le nom du bot et le nom d'utilisateur. Vous verrez alors votre jeton de bot Telegram.

<Image src="/images/integrations/telegram-app2.png"
width="718"
height="404"
alt="Image"
/>
3. Copiez votre jeton de bot Telegram dans le presse-papiers
4. Pour personnaliser votre bot Telegram, entrez "/mybots", sélectionnez votre bot et cliquez sur "Edit bot" bouton.

<Image src="/images/integrations/telegram-set.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>
  Dans ce panneau, vous pouvez définir le nom, la description et l'image du Bot.
## Étape 2 : lancez votre bot Telegram

1. Chaque chatbot créé sur Ollabot possède un code d'intégration unique pour l'intégration. Une fois votre chatbot configuré sur Ollabot, accédez à votre page de tableau de bord et sélectionnez le bot spécifique que vous souhaitez intégrer à votre application Bubble.

<Image src="/images/integrations/dashboard.png"
width="718"
height="404"
alt="Image"
/>
2. Cliquez sur le bot pour accéder à sa page d'aperçu. De là, accédez à l'onglet « Intégrer sur le site » et cliquez sur le bouton « Rendre public ».

3. Une fenêtre contextuelle apparaîtra avec le code HTML fourni. Cliquez sur « Copier le script » pour copier l'extrait de code.

<Image src="/images/integrations/embed.png"
width="718"
height="404"
alt="Image"
/>

4. Sur votre tableau de bord, sélectionnez le bouton "Lancer l'éditeur" à côté.

<Image src="/images/integrations/telegram-embed.png"
width="718"
height="404"
alt="Image"
/>


6. Testez votre télégramme pour tester votre chatbot.

<Image src="/images/integrations/telegram-test.png"
width="718"
height="404"
alt="Image"
/>

Félicitations, votre chatbot Ollabot est désormais en ligne sur votre application Telegram !
