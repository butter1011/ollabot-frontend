## Étape 1: Configurer Votre Chatbot Ollabot

Pour intégrer votre chatbot AI Ollabot dans votre application Framer, commencez par vous connecter à votre compte Ollabot. Si vous n'avez pas encore de compte, vous pouvez vous inscrire gratuitement. Après vous être connecté, configurez votre bot sur la plateforme Ollabot en téléchargeant des sources de données pertinentes telles que des fichiers, des extraits de texte, des sites web ou des paires de questions-réponses pour construire la base de connaissances du chatbot.

Suivez cette feuille de route étape par étape pour déployer avec succès votre chatbot Ollabot.

## Étape 2: Générer et Copier le Code d'Intégration de Votre Chatbot Ollabot

1. Chaque chatbot créé sur Ollabot dispose d'un code d'intégration unique. Une fois votre chatbot configuré sur Ollabot, accédez à votre page de tableau de bord et sélectionnez le bot spécifique que vous souhaitez intégrer à votre application Framer.

<Image src="/images/integrations/dashboard.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>
2. En cliquant sur le bot, vous serez redirigé vers sa page de prévisualisation. À partir de là, allez à l'onglet "Intégrer sur le Site" et cliquez sur le bouton "Rendre Public".

3. Une fenêtre contextuelle apparaîtra avec le code HTML fourni. Cliquez sur "Copier le Script" pour copier le snippet de code.

<Image src="/images/integrations/embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

## Étape 2: Connectez-vous à Votre Site Web Framer et Intégrez Votre Chatbot

1. Connectez-vous à votre site web Framer et accédez à votre tableau de bord de projet.

2. Cliquez sur le projet de site web sur lequel vous souhaitez intégrer votre chatbot Ollabot.

3. Ouvrez le projet de site web, localisez la page sur laquelle vous souhaitez intégrer votre chatbot.

4. Cliquez sur l'icône Plus en haut, puis allez à l'utilitaire sous éléments et sélectionnez "Intégrer".

<Image src="/images/integrations/framer-click.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

5. Sous la section HTML, collez votre code d'intégration Ollabot dans la boîte et votre popup de chatbot devrait apparaître.

<Image src="/images/integrations/framer-embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

6. Si tout est fait correctement, vous devriez maintenant voir une icône de chatbot Ollabot flottante dans le coin inférieur gauche de votre site web Framer (sur la page que vous y avez ajoutée).

<Image src="/images/integrations/framer-chat.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

Félicitations ! Votre chatbot Ollabot est maintenant opérationnel sur votre site web Framer.

**Remarque** : Vous pouvez personnaliser l'apparence et les couleurs de votre chatbot sur le tableau de bord Ollabot. Accédez à votre tableau de bord, sélectionnez le bot, cliquez sur l'onglet "Paramètres" en haut de la page, puis cliquez sur "Interface de Chat" dans la barre latérale gauche pour accéder aux options de personnalisation du chatbot.
