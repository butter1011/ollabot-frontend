## Étape 1: Configuration de Votre Chatbot Ollabot

Pour intégrer votre chatbot AI Ollabot dans votre site WordPress, commencez par vous connecter à votre compte Ollabot. Si vous n'avez pas encore de compte, vous pouvez vous inscrire gratuitement. Après vous être connecté, configurez votre bot sur la plateforme Ollabot en téléchargeant des sources de données pertinentes telles que des fichiers, des extraits de texte, des sites web ou des paires de questions-réponses pour construire la base de connaissances du chatbot.

Suivez cette feuille de route étape par étape pour déployer avec succès votre chatbot Ollabot.

## Étape 2 : Générez et copiez votre code d'intégration du chatbot Ollabot

1. Chaque chatbot créé sur Ollabot dispose d'un code d'intégration unique. Une fois votre chatbot configuré sur Ollabot, accédez à votre page de tableau de bord et sélectionnez le bot spécifique que vous souhaitez intégrer à votre application Bubble.

<Image src="/images/integrations/dashboard.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>
2. En cliquant sur le bot, vous accédez à sa page de prévisualisation. De là, allez dans l'onglet "Intégrer sur le site" et cliquez sur le bouton "Rendre public".

3. Une fenêtre pop-up apparaîtra avec le code HTML fourni. Cliquez sur "Copier le script" pour copier le fragment de code.

<Image src="/images/integrations/embed.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

## Étape 3 : Installez Ollabot sur votre site WordPress

1. Connectez-vous à votre tableau de bord administrateur WordPress :

L'URL de votre tableau de bord est généralement `votrenomdedomaine.com/wp-admin/`. Vous pouvez également y accéder via votre panneau de contrôle d'hébergement web.

2. Cliquez sur Personnaliser : Dans la barre de navigation supérieure de votre tableau de bord administrateur WordPress, cliquez sur `Personnaliser`.

<Image src="/images/integrations/wordpress-customize.png" 
  width="718" 
  height="404" 
  alt="Image" 
/>

3. Cliquez sur `Script personnalisé` : Dans la barre latérale gauche du tableau de bord administrateur WordPress, cliquez sur `Script personnalisé`.

<Image src="/images/integrations/wordpress-script.png" 
  width="404" 
  height="718" 
  alt="Image" 
/>

4. Copiez et collez le script d'intégration de votre chatbot :

<Image src="/images/integrations/wordpress-embed.png" 
  width="404" 
  height="718" 
  alt="Image" 
/>

5. Enregistrez vos modifications :

   - Cliquez sur "Enregistrer les modifications". Votre chatbot Ollabot devrait maintenant être en ligne sur votre site WordPress !
